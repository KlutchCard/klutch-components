# klutch-components

Klutch Components for MiniApps

## Installation

```sh
npm i @klutch-card/klutch-components
```

## Usage

```js
import { KlutchComponents } from "@klutch-card/klutch-components"

// ...

const result = await KlutchComponents.multiply(3, 7);
```
